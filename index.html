Pelajari Materi
WEB DEVELOPMENT

Kombinasi Tiga Pilar Web

Website yang hebat menggabungkan struktur, keindahan gaya visual, dan interaksi logika yang hidup.

HTML (HyperText Markup Language)

HTML adalah tulang punggung atau struktur dasar website. HTML menentukan letak tombol, teks, gambar, atau video menggunakan sistem tag.

CSS (Cascading Style Sheets)

CSS adalah baju atau dekorasi visual website. Mengatur warna, ukuran font, layout responsif, hingga efek pendaran neon yang estetik.

JavaScript (JS)

JavaScript adalah sistem saraf atau otak yang memberikan kehidupan interaktif, seperti memproses kalkulasi dan validasi logika.

PYTHON BASICS

Sintaks Dasar Python

Python adalah bahasa pemrograman populer yang sangat disukai karena struktur kodenya bersih dan mudah dipahami layaknya bahasa Inggris.

Fungsi print()

Digunakan untuk menampilkan teks atau data ke layar terminal. Contoh: print("Hello Nexa").

Variabel & Tipe Data

Wadah memori untuk menyimpan data sementara. Seperti teks (String) ataupun angka bulat (Integer).

Logika Kondisional (if-else)

Mengatur jalannya alur program komputer berdasarkan evaluasi kondisi benar atau salah.

MOBILE APPS

Mobile Apps Development

Merancang dan membangun aplikasi interaktif yang dikhususkan berjalan di perangkat smartphone maupun tablet.

Dua Ekosistem Raksasa

Aplikasi dikembangkan untuk platform Android (didukung Google) dan platform iOS (ekosistem Apple).

Native vs Cross-Platform

Native dibuat khusus untuk satu sistem operasi, sementara Cross-Platform (seperti Flutter atau React Native) memungkinkan satu kode berjalan di kedua OS sekaligus.
